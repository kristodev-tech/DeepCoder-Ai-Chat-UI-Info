🚀 DeepCoder AI Chat UI

A Developer-Focused, Fully Featured AI Chat Assistant Powered by Ollama

📄 Description

DeepCoder AI Chat UI offers a feature-rich, customizable chat experience for developers, creators, and tech enthusiasts. Built using Flask and Ollama, it integrates powerful models like DeepSeek Coder V2, DeepSeek V2, and Codellama Meta — providing a modern alternative to ChatGPT with enhanced functionality and mobile support.

✨ Key Features

  • 💬 Interactive AI chat assistant for technical questions
  • 🧠 Streaming responses with auto-scroll
  • 📝 Inline edit and resend
  • 📦 Download as PDF/Markdown/Text
  • 🔀 Model switching
  • 🌙 Dark mode, beautiful UI
  • 🚮 Session management
  • 📊 Token usage display

📊 Model Information

Primary: DeepSeek Coder V2 (Q4_0)
Additional: DeepSeek V2, Codellama Meta
